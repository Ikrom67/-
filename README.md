# english-maktabi

онлайн школа для изучения  английского языка

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the server:
   ```bash
   npm run dev
   ```

3. Visit http://localhost:3000 and check /health for DB connectivity.

## Config values

- Organization: avtomat
- Project name: english-maktabi
- Region: Frankfurt (EU) or Singapore
- Database password: (stored in .env as an example)
